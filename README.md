# 招标采购人工具箱 (Tender & Procurement Toolbox)

## 1. 项目简介

本项目集成了招标采购过程中最常用的6款辅助工具，旨在为招标人、代理机构及相关从业人员提供便捷的费用测算与流程辅助服务。所有计算逻辑均严格基于国家及湖南省相关最新政策文件开发。

## 3. 功能亮点

- **离线可用**：所有逻辑封装在单个 HTML 文件中，下载后无需网络即可运行。
- **一键切换**：集成式面板，快速在不同计算器间切换。
- **数据导入**：随机抽取系统支持 Excel (`.xlsx`, `.xls`) 格式名单批量导入。
- **即时计算**：输入金额后实时显示结果及详细计算过程（如差额定率累进的各段明细）。

## 4. 免责声明

1. 本工具箱的计算结果仅供参考，不作为最终法律依据或结算凭证。
2. 在实际业务中，请以各省市最新发布的官方红头文件、招标文件条款及双方签订的合同约定为准。
3. 政策文件可能会随时间更新或废止，请使用者注意核对依据文件的时效性。
