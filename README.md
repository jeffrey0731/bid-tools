# 全能招标采购工具箱 (专业完整版)

**One-Stop Tendering & Procurement Toolkit - Professional Edition**

## 1. 项目简介

本项目是专为招标人、造价工程师、招标代理机构及评标专家打造的**全能型辅助工具集**。

在原 v2.4 版本的基础上，本版本进行了底层架构重构与功能深度整合。摒弃了复杂的编译依赖，采用**原生 JavaScript (Vanilla JS)** 开发，将 **12 款核心工具**整合至**单个 HTML 文件**中。界面升级为更加专业的侧边栏导航布局，并完整保留了各专业领域的深度计算逻辑（如差额定率累进、复杂的行业调整系数、专家抽取回避逻辑等）。

## 2. 核心特性

- **架构升级**：移除 React 依赖，改为轻量级原生 JS，兼容性更强，打开即用，无编译门槛。
- **深度整合**：将《工程造价收费计算器》与《专家抽取系统 GM 版》完美融合，实现数据互通与统一管理。
- **专业版面**：采用左侧导航 + 右侧功能区的响应式布局，操作更符合桌面办公习惯。
- **功能全面**：覆盖从前期咨询、设计、造价、监理到招投标实施、专家抽取、费用结算的全生命周期。

## 4. 使用说明

### 4.1 运行方式
1.  下载 `全能招标工具箱.html` 文件。
2.  双击文件，使用任意现代浏览器（Chrome, Edge, Firefox, Safari）打开即可。
3.  **注意**：由于工具箱引用了 Tailwind CSS、FontAwesome 和 PDF.js 的 CDN 资源，**初次打开或使用特定功能（如发票解析）时需要保持网络连接**。

### 4.2 专家抽取系统数据格式
如需使用“专家抽取系统”导入数据，请准备 **CSV (逗号分隔)** 格式文件（推荐使用 Excel 另存为 CSV UTF-8）：
- **政府采购库**：第1行为标题，第2行起为数据。第4列(D列)开始为专业名称（分号或换行分隔）。
- **综合库**：第1-2行为标题，第3行起为数据。第4列(D列)开始为专业名称。
- 系统支持 **GBK** 与 **UTF-8** 编码切换，防止乱码。

### 4.3 打印设置
本系统针对打印进行了深度优化（特别是专家签到表和发票打印）：
- 打印时，侧边栏会自动隐藏。
- 请在浏览器打印设置中，将 **边距** 设置为 **“无”** 或 **“最小值”** 以获得最佳效果。
- 需勾选 **“打印背景图形”** 以显示表格底色。
